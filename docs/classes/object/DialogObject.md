# `DialogObject` 类级分析

## 1. 类职责

`DialogObject` 表示游戏中的对话框或剧情提示框。

它常用于：

- 教程提示
- 任务说明
- 剧情对白
- 操作反馈

## 2. 核心字段

- `speakerName`
- `message`
- `fontSize`

## 3. 行为

### `setMessage()`

改变当前显示文本。

### `setSpeakerName()`

改变说话者名字。

### `render()`

绘制对话框及其文字内容。

## 4. 文本排版

对话框会自动换行。

这对中文长句尤其重要，因为游戏里很多提示都不会只占一行。

## 5. 与状态机的关系

对话框经常与 `DefaultGameStateMachine` 联动：

- 菜单选择结果会同步到对话框
- 暂停时可以隐藏当前对话框
- 恢复后可以还原对话框

## 6. 设计价值

它让游戏可以很轻松地加入教程、剧情和交互提示，而不需要额外开独立 UI 系统。
