# `ProjectileObject` 类级分析

## 1. 类职责

`ProjectileObject` 是所有投射物的统一实现。

它支持多种弹药行为：

- 普通子弹
- 照明弹
- 爆破弹
- 穿透弹
- 追踪弹
- 激光弹
- 追猎弹

## 2. 关键状态

- `velocityX` / `velocityY`
- `damage`
- `projectileType`
- `gravityAffected`
- `explosive`
- `explosionRadius`
- `fuseSeconds`
- `lightRadius`
- `homingStrength`
- `pierceTargetsRemaining`
- `age`
- `lifetime`
- `exploding`

## 3. 创建方式

### 普通构造

适合标准子弹。

### `createBomb()`

适合爆破弹，带：

- 爆炸半径
- 爆炸伤害
- 引信时长

## 4. 更新流程

### 4.1 生命周期

每帧会累计年龄并减少剩余寿命。

寿命到期后：

- 普通弹：直接失活
- 爆炸弹：触发爆炸

### 4.2 重力

如果弹种受重力影响，会持续下坠。

### 4.3 追踪

追踪类弹体会逐帧调整方向，让运动轨迹更接近目标。

### 4.4 碰撞判定

弹体会与世界中其他对象碰撞：

- 打到角色：造成伤害
- 打到可破坏场景：破坏
- 打到固体墙体：失活
- 打到障碍：可能触发爆炸

## 5. 友伤与穿透

投射物支持：

- 是否能伤害队友
- 是否可穿透多个目标
- 已命中的目标记录

这让子弹系统比“碰到就删”复杂得多。

## 6. 爆炸

爆破弹会：

1. 进入爆炸状态
2. 对范围内实体造成伤害
3. 触发屏幕震动
4. 播放爆炸音效
5. 生成烟雾/爆炸视觉

## 7. 渲染差异

不同弹种有不同画法：

- 标准弹：普通圆点
- 照明弹：带光晕
- 激光弹：细长高亮
- 追猎弹：带尾迹
- 炸弹：带引线和爆炸过程

## 8. 游戏意义

这个类把“武器行为”从玩家和怪物里解耦出来，使战斗系统可以通过弹种扩展，而不必修改每个角色逻辑。
