# Audio attribution

The following sound effects were downloaded from Wikimedia Commons and stored in
`src/main/resources/audio/`.

## Files

- `hurt.wav`
  - Wikimedia Commons file page: https://commons.wikimedia.org/wiki/File:Hurt2.wav
  - Direct audio URL: https://upload.wikimedia.org/wikipedia/commons/7/75/Hurt2.wav
  - Commons page lists the source as Freesound / thecheeseman and provides the
    license details on the file page.

- `damage.wav`
  - Wikimedia Commons file page: https://commons.wikimedia.org/wiki/File:Damage_taken_%28Gravity_Sound%29.wav
  - Direct audio URL: https://upload.wikimedia.org/wikipedia/commons/0/03/Damage_taken_%28Gravity_Sound%29.wav
  - Used for the player taking damage / damage prompts.

  - Commons page lists the source as Freesound / LloydEvans09 and the license
    details on the file page.

- `heal.wav`
