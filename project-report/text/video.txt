================================================================================
                    HNSF Game Engine Demo 答辩视频制作指南
                    《软件设计实作》课程 - 10分钟展示方案
================================================================================

【总时长：约10分钟】
【核心目标：展示游戏玩法 + 引擎技术能力】
【录制策略：分段子视频录制，后期剪辑合成】

================================================================================
                          第一部分：开场引入
================================================================================

■ 子视频 1：项目介绍与精彩预览 [时长：60秒]

  展示内容：
  - 项目标题画面/HNSF Game Engine Logo
  - 一句话介绍："基于Java Swing的2D游戏引擎原型"
  - 主菜单界面展示
  - 快速剪辑3-4个精彩画面预览：
    ① 程序化生成的地形全景
    ② 角色与怪物战斗场景
    ③ 到达终点过关画面

  关卡选择：主菜单界面 + 多个精彩片段

  操作要点：
  - 启动游戏，停留在标题/菜单画面约5秒
  - 各精彩片段每个约5秒，后期拼接

================================================================================
                          第二部分：角色操控展示
================================================================================

■ 子视频 2：基础移动与跳跃 [时长：90秒]

  展示内容：
  - 使用W/A/S/D或方向键控制角色四向移动
  - 展示流畅的加速与减速过程（惯性手感）
  - 按空格键跳跃，展示只有贴近地面时才能起跳
  - 连续跳上多个平台
  - 从高处落下展示重力加速度
  - 碰撞与阻挡效果

  关卡选择：有平台高差的关卡

  操作要点：
  - 先在平坦地面左右移动，展示加速/减速
  - 连续跳跃上平台
  - 从高台跳下展示下落加速度
  - 撞墙展示碰撞阻挡

--------------------------------------------------------------------------------

■ 子视频 3：冲刺、体力与颜色切换 [时长：60秒]

  展示内容：
  - 体力条UI位置展示
  - 快速移动消耗体力，停止后自动恢复
  - 按C键循环切换角色颜色（每个颜色停留2秒）
  - 接近互补色方块时受伤害
  - 受伤后1秒无敌闪烁效果

  关卡选择：包含不同颜色方块的关卡

  操作要点：
  - 持续移动展示体力消耗和恢复
  - 依次切换展示不同颜色
  - 故意接触互补色方块展示伤害机制

================================================================================
                          第三部分：战斗系统展示
================================================================================

■ 子视频 4：射击攻击与击退效果 [时长：45秒]

  展示内容：
  - 按K键发射投射物
  - 投射物飞行轨迹
  - 射击冷却时间展示（快速连按展示限制）
  - 投射物命中怪物产生击退效果
  - 连续命中叠加击退

  关卡选择：开阔无障碍关卡 + 有史莱姆的区域

  操作要点：
  - 连续射击展示投射物
  - 改变方向射击
  - 对怪物射击展示击退

--------------------------------------------------------------------------------

■ 子视频 5：怪物遭遇展示 [时长：90秒]

  展示内容：
  - 史莱姆：绿色软体，移动缓慢但跳跃力强
  - 蜘蛛：深褐色，速度最快，主动追踪
  - 蝙蝠：紫色飞行单位，空中盘旋俯冲
  - 幽灵：半透明飘浮体，移动诡异
  - 石像鬼：灰色坚硬，生命值高
  - 飞龙：橙红色精英，攻击欲望强
  - 击杀怪物获得经验值
  - 被攻击受伤闪烁效果

  关卡选择：包含多种怪物的关卡

  操作要点：
  - 每种怪物展示约10-15秒
  - 重点展示行为差异
  - 击杀展示经验获得
  - 故意被攻击展示受伤效果

--------------------------------------------------------------------------------

■ 子视频 6：击杀掉落与奖励 [时长：30秒]

  展示内容：
  - 击杀怪物后掉落治疗道具
  - 自动拾取掉落物
  - 生命值恢复效果

  关卡选择：有掉落配置的怪物关卡

  操作要点：
  - 击杀怪物等待掉落
  - 接近自动拾取
  - 展示生命值恢复

================================================================================
                          第四部分：道具系统展示
================================================================================

■ 子视频 7：道具收集展示 [时长：60秒]

  展示内容：
  - 绿色生命道具：拾取恢复HP
  - 蓝色速度道具：拾取后移动速度提升
  - 亮黄色光球：视野范围扩大（暗区对比）
  - 金色/橙色金币宝石：获得经验值

  关卡选择：放置了各类道具的关卡（含暗区）

  操作要点：
  - 先受伤后拾取生命道具展示恢复
  - 拾取速度道具前后移动对比
  - 在暗区拾取光球展示视野扩展
  - 连续拾取金币展示经验累积

================================================================================
                          第五部分：光照与物理系统展示
================================================================================

■ 子视频 8：动态光照系统 [时长：60秒]

  展示内容：
  - 正常亮度下的场景
  - 进入暗区，环境光降低
  - 角色移动时光源跟随
  - 投射物在暗区飞行的光源效果
  - 金色传送门终点的脉动光芒
  - 粒子环绕动画

  关卡选择：有明暗区域变化 + 包含终点的关卡

  操作要点：
  - 先展示正常亮度
  - 进入暗区展示光照效果
  - 在暗区射击展示投射物光源
  - 从远到近接近终点展示光芒效果

--------------------------------------------------------------------------------

■ 子视频 9：物理引擎综合展示 [时长：45秒]

  展示内容：
  - 重力作用：从高处落下加速
  - 地面检测：跳跃时机判断
  - 碰撞响应：撞墙/落地效果
  - 平台边缘精确判定
  - 弹跳效果（如有）

  关卡选择：有高台和平台的关卡

  操作要点：
  - 从高台跳下展示重力
  - 连续跳上跳下平台
  - 展示平台边缘精确判定

================================================================================
                          第六部分：关卡与通关展示
================================================================================

■ 子视频 10：程序化生成地形 [时长：45秒]

  展示内容：
  - 启动新游戏时的地形生成
  - 基于柏林噪音的地形起伏
  - 自动生成的森林植被和洞穴结构
  - 摄像机快速移动展示全景

  关卡选择：重新生成的新关卡

  操作要点：
  - 启动新游戏展示生成过程
  - 快速移动展示地形全貌
  - 展示不同地形区域特色

--------------------------------------------------------------------------------

■ 子视频 11：关卡胜利与失败 [时长：45秒]

  展示内容：
  - 穿越关卡到达金色传送门终点
  - 接触终点触发胜利
  - 结算界面展示
  - 生命值归零触发失败
  - 失败原因显示
  - 重生机制展示

  关卡选择：可快速到达终点的关卡

  操作要点：
  - 完成一局游戏展示胜利
  - 展示结算画面
  - 演示失败和重生流程
